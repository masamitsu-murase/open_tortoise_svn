Example

Prepare

  1. Set "Path to TortoiseProc.exe".
  2. Add a URL "https://github.com/masamitsu-murase/open_tortoise_svn.git/branches/firefox_webextensions".
  3. Set default action to "Repository Browser".
  4. Add extension actions.
    • "*.md" as "Log"
    • "*.json, *.js" as "Blame"
    • "*.cpp" as "Open in Firefox"
  5. Check "A help tip".
